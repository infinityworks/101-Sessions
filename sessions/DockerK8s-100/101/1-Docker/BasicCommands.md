# Basic Commands
## List Docker CLI commands
```
docker
docker container --help
```
## Display Docker version and info
```
docker --version
docker version
docker info
```

## Execute Docker image
```
docker run hello-world
```

## List Docker images
```
docker image ls
```

## List Docker containers (running, all, all in quiet mode, shorthand)
```
docker container ls
docker container ls --all
docker container ls -aq
docker ps -a
```
